# Frontend

## Funcionalidades

- Cadastro de um novo aluno
- Exibição de todos alunos cadastrados
- Gráfico 'pizza' contento a quantidade de alunos de cada sexo
- Mapa com Markers que são centralizados com o clique

## Utilização

1. npm i
2. npm run dev
